Requirimientos
MySQL Server 
Node.js

ejecutar archivo scriptdb.sql para construir base de datos MySQL


Ingresando a la carpeta del proyecto es necesario instalar librerias
npm install

ejecutar en siguiente comando para iniciar la aplicacion

node app.js
